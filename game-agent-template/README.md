# 项目名称

> 占位。复制到新项目后，请用真实游戏名称和对外说明替换本文件。

## 这是什么

先用一句话概括这款游戏最核心的卖点，再用 1 到 2 段说明游戏类型、目标平台、玩家扮演的角色、主要玩法循环，以及玩家为什么会想继续玩下去。

## 核心体验

- 体验点 1：一句话描述玩家最常进行的关键操作。
- 体验点 2：一句话描述最重要的情绪反馈或乐趣来源。
- 体验点 3：一句话描述区别于同类游戏的设计特征。
- 体验点 4：一句话描述当前版本最值得体验的内容。

## 当前状态

- 当前阶段：原型 / 玩法设计 / 内部试玩 / 公开测试 / 正式发售
- 当前版本：`vX.Y.Z`
- 目标平台：PC / 主机 / iOS / Android / Web
