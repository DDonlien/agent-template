## 这是什么

> 一句钩子（the hook）。
> 例如：一具焦尸证明他死在家里。一串记录证明他逃了出去。请提交你的保险审核报告。

接着用一段话说明：游戏类型、平台、玩家身份、时代设定、核心循环、玩家在游戏里做什么。

## 核心体验

- **体验点 1**：一句话描述玩家会做什么、感受到什么。
- **体验点 2**：一句话。
- **体验点 3**：一句话。
- **体验点 4**：一句话。

## 关键设计

- **核心交付物**：一句话说明玩家最终产出什么。
- **玩家身份**：例如第三方理赔审查员（不是侦探 / 警察 / 家属）。
- **时代与世界观**：例如架空 2002 年、某城某小区。
- **核心循环**：阅读材料 → 关键词搜索 → 收集证据 → 报告书归类 → 提交结论。
- **世界钩子**：玩家在哪一节点会接触"状态冲突"现象；这暗示主 IP 的什么主题。

## 状态

- 当前案 / 关卡 / Demo：例如第一案 - 理赔单 1106（南栖公寓）。
- 当前阶段：世界观 / 玩法设计 / 内部试玩 / 公开测试 / 正式发售。
- 目标平台：iOS / Android / Steam / Switch 等。
